# Задание 9
### На HQ-SRV настройте программное обеспечение fail2ban для защиты ssh
### 9.1 Укажите порт ssh
### 9.2 При 3 неуспешных авторизациях адрес атакующего попадает в бан
### 9.3 Бан производится на 1 минуту

*Настройка HQ-SRV*
```bash
apt-get install fail2ban -y
systemctl enable --now fail2ban.service
mcedit /etc/fail2ban/jail.local


[sshd]
enabled = true
filter = sshd
action = nftables[name=SSH, port=2026, protocol=tcp]
logpath = /var/log/auth.log
findtime = 600
maxretry = 3
bantime = 60
backend = systemd


systemctl restart fail2ban.service
systemctl status fail2ban.service
```
